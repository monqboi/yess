Conclusion

สรุปผลจากกิจกรรม
ผ่านการเข้าร่วมโครงการค่ายฝึกอบรมเชิงปฏิบัติการวิทยาการคอมพิวเตอร์ ครั้งที่ 14 ในรูปแบบออนไลน์ ซึ่งจัดโดยคณะวิทยาศาสตร์ ภาควิชาวิทยาการคอมพิวเตอร์ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารบาดกระบัง เป็นอีกหนึ่งค่ายที่ทำให้ฉันได้ฝึกฝนและคุ้นชินกับภาษาโปรแกรมมากขึ้น ได้ตกตะกอนถึงความชอบและความสามารถของตนเองว่าชื่นชอบในภาษา HTML และ CSS แต่ก็ได้เปิดใจและรู้จักกับภาษา Java มากขึ้นในการทำ Mini Project ถึงแม้จะภาษาที่ไม่ถนัดแต่เพื่อนในทีมและพี่ ๆ ประจำบ้านก็ซัพพอร์ตกันอย่างดี คอยสอนในส่วนที่ไม่รู้จนทำให้ได้รู้และไขข้องใจ นับว่าหลังจากผ่านค่ายนี้ไปนอกจากได้ความรู้ที่หลากหลายภาษาแล้วนั้น ยังได้เพื่อนได้ความสัมพันธ์ได้มิตรภาพที่หลากหลายและน่ารักอีกด้วย
