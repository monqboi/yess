เข้าร่วมการประกวดโครงงานคณิตศาสตร์ประยุกต์ ระดับมัธยมศึกษาตอนปลาย ประจำปี 2565
ในงานมหกรรมวิทยาศาสตร์และเทคโนโลยีแห่งชาติ ณ อิมแพ็ค เมืองทองธานี โดยผลงานที่นำเสนอ

นำภาษา Python มาประยุกต์ใช้ในการแข่งขันครั้งนี้

Introduction

ที่มาของโครงงาน
Calcudator ได้รับแรงบันดาลใจจากความสามารถของผู้ที่มีอาการออทิสติกซึ่งสามารถคำนวณวันในสัปดาห์ที่กำหนดได้อย่างแม่นยำและรวดเร็ว คณะผู้จัดทำจึงเกิดความสนใจที่ว่า หากเป็นบุคคลทั่วไปจะสามารถคำนวณวันในสัปดาห์ได้เฉกเช่นเดียวกันหรือไม่ ซึ่งได้รับรางวัลชมเชยการประกวดโครงงานคณิตศาสตร์ประยุกต์รอบชิงชนะเลิศ ในงานมหกรรมวิทยาศาสตร์และเทคโนโลยีแห่งชาติ จัดโดย มูลนิธิส่งเสริมอัจฉริยภาพทางคณิตศาสตร์ ร่วมกับ ภาควิชาคณิตศาสตร์ คณะวิทยาศาสตร์ประยุกต์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ ซึ่งจากการแข่งขันนี้ได้เสริมสร้างความรู้ความเข้าใจในเรื่องของความรู้ทางคณิตศาสตร์และมอดูลใน Python รวมไปถึงความสามารถในการแก้ไขปัญหาและฝึกการทำงาน เป็นทีมเป็นอย่างมาก

บทบาทหน้าที่
รับผิดชอบ Graphic Design ในส่วนของหน้าปกและแผ่นพับ รวมไปถึงสูตรการหาวันแรกของปี พร้อมทั้งร่วมกันกับสมาชิกประยุกต์สูตรการคำนวณที่คิดได้เข้ากับ Python โดยในอนาคตมีเป้าหมายที่จะศึกษาเพิ่มเติมในเรื่องของ ux/ui และ front-ed เพื่อพัฒนาออกมาในรูปแบบของเว็บไซต์หรือแอปพลิเคชัน เพื่ออำนวยความสะดวกให้แก่ผู้ใช้งาน
